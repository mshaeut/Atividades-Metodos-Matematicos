bolzano_weiertrass - Escreva um programa e envie o arquivo fonte, ou um link do mesmo, com a seguinte capacidade: dados um número real com L ∈ [-1, 1], um número natural Q, e uma tolerância ϵ > 0 pequena, encontra Q
subíndices nk da sequência (cos(n)) tal que |ank − L| < ϵ. Liste os pares (nk, ank) e ilustre-os graficamente numa mesma figura, para k = 1, 2, ..., Q.
