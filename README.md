**Instruções recebidas para realizar a implementação dos arquivos**:

**bolzano_weiertrass_convergencia_de_sequencias** - Escreva um programa e envie o arquivo fonte, ou um link do mesmo, com a seguinte capacidade: dados um número real com L ∈ [-1, 1], um número natural Q, e uma tolerância ϵ > 0 pequena, encontra Q
subíndices nk da sequência (cos(n)) tal que |ank − L| < ϵ. Liste os pares (nk, ank) e ilustre-os graficamente numa mesma figura, para k = 1, 2, ..., Q.

**operadores_lineares_autovalores_autovetores** - Fazer um programa, preferencialmente em Python usando algumas bibliotecas tais como “linalg e matplotlib” com as seguintes capacidades:
Dada uma matriz quadrada Am×m o programa calcula todos os autovalores de A, indica a multiplicidade de cada um e calcula os autovetores unitários associados a cada um dos autovalores. Em seguida, para os casos em que m = 2 e m = 3 o programa plota cada um dos autovetores unitários (com uma cor associada a cada um dos autovalores). Além disso, permite ao usuário escolher um dos autovetores (digamos vaut) e fornecer as coordenadas de um outro vetor (digamos vdiraut) na direção do autovetor vaut escolhido. Por fim, o programa plota a imagem A vdiraut com a mesma cor escolhida para vaut já plotado e deixa o usuário livre para outras escolhas.

**series_numericas** - Partindo de uma função que defina o termo geral de uma sequência a(n), para n = n0, n0 + 1, n0 + 2, · · · , implemente um programa para ilustrar o comportamento da série numérica P∞ n=n0 a(n) = P∞ n=0 a(n0 + n), com as duas opções exclusivas de execução:
(a) o usuário sabe que a série é convergente para a soma S;
(b) o usuário não sabe se a série é convergente.
